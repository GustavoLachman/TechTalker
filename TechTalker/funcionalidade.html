Como funciona

O TechTalker visa ajudar as pessoas acometidas de Afasia de Broca a se comunicar, utilizando a interface gráfica desenvolvida por nós da TechTalker. Está interface de fácil uso e adapitativa para o uso de cada usuário, sendo assim a alternativa ideal para o paciente com a condição voltar a se comunicar com facilidade e comodidade.

Usabilidade

O portador de Afasia de Broca, consegue raciocinar mas não consegue se expressar com palavras, nesta hora que o TechTalker ajuda-lo, com o nosso software ele podera atraves de uma interface gráfica utilizar imagens pré configuradas e/ou modificadas pelo usuário, com isso ele pode salvar imagens de pessoas próximas por exemplo.
